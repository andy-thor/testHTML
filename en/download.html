Clone from GitHub repository

Clone the repository

$ git clone https://github.com/Andy-thor/StickMan.git

Enter the project folder

$ cd stickman-master


You can install it...

$ sudo python setup.py install

...or run directly without installing(Linux)

$ python stickman
