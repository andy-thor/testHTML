Clonar repositorio de GitHub

Clonar el repositorio

$ git clone https://github.com/Andy-thor/StickMan.git

Entrar a la carpeta del proyecto

$ cd stickman-master


Puedes instalarlo...

$ sudo python setup.py install

...o ejecutar directamente sin instalarlo(Linux)

$ python stickman
